8.2 pivot_wider()

The pivot_wider() function reverses what pivot_longer() does, turning long datasets into wide formats. Suppose you want to revert our eagle_nests_longer data back to its original shape:

eagle_nests_longer |>
    pivot_wider(
        names_from = "year",
        values_from = "nests"
    )
1
pivot_wider() transforms long data into a wider format.
2
names_from identifies which column contains the new column names.
3
values_from identifies the column whose values will fill the new cells.

Letâ€™s see a more practical example with our penguins dataset. Perhaps we want to compare bill length between male and female penguins for each species. We can create a summary of the mean bill length by species and sex, then convert that summary into a wide layout:
